# 🌿 Mood Journal

A minimal, beautiful daily mood tracker that lives entirely in your browser — no backend, no login, just you and your thoughts.

## ✨ Features

- **5 mood states** — Happy, Energized, Meh, Anxious, Sad
- **Daily note** — jot down what's on your mind
- **7-day mood chart** — color-coded weekly overview at a glance
- **Persistent storage** — entries saved locally via `localStorage`
- **Delete entries** — clean up anytime
- **Zero dependencies** — pure HTML, CSS, JavaScript

## 🚀 Getting Started

No setup needed! Just open the file in your browser:

```bash
# Clone the repo
git clone https://github.com/keerthana-badveli/mood-journal.git

# Open in browser
cd mood-journal
open index.html   # macOS
start index.html  # Windows
```

Or simply [download the HTML file](./index.html) and open it directly.

## 💡 How It Works

1. Select your mood for the day
2. Optionally add a short note
3. Hit **Save today's mood**
4. View your weekly pattern in the 7-day chart
5. Scroll down to revisit past entries

All data is stored in your browser's `localStorage` — nothing leaves your device.
